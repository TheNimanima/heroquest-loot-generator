## HOW TO USE THIS

1. **For ✓ items:** scan through the text — if anything's wrong, photograph the card and we'll correct it
2. **For ✗ items:** photograph each card; doesn't need to be perfect, just legible
3. **Bulk approach:** if it's faster, lay 5-10 cards out at a time, photograph the spread, send the image. I can read multiple cards from a single photo.
4. **Priority order if you don't want to do all at once:**
   - High value (most missing): Wizards of Morcar, Rise of the Dread Moon, Jungles of Delthrak, Knight Collection
   - Medium value: Prophecy of Telor, Spirit Queen's Torment, Crypt of Perpetual Darkness, Monk Collection
   - Low value (small gaps): base game Mythic artifacts, Kellar's Keep equipment, Mage of the Mirror remainder
